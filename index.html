Welcome to my CSCE 3530 Web Server

About This Project

This web page is served by a custom C-based HTTP server that I built for CSCE 3530 – Computer Networks at UNT.

It responds to browser requests, delivers HTML pages, returns a custom 404 page when a file is missing, and cleanly shuts down if the exit.html page is requested.

Why this is so Cool!

  • Demonstrates hands-on experience with low-level socket programming, HTTP protocol basics, and concurrency fundamentals.
  • Showcases my ability to solve real-world networking tasks without relying on high-level frameworks or external code.

Whether you’re browsing from campus or connecting from elsewhere, this lightweight server handles your requests and logs the activity right to the console for easy debugging and demos.

Next Steps

Try requesting other pages like test.html to see the 404 error, or exit.html to gracefully shut down the server.
